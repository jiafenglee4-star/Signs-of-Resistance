## 📝 项目简介 / Project Overview

### 中文
《Signs of Resistance》是一款将沉浸式叙事与手语（ASL）交互相结合的 HTML5 游戏。在这个项目中，玩家不仅仅是在屏幕前点击鼠标，而是需要通过真实的手语动作来推动剧情的发展。

我设计这个项目的初衷是希望通过技术手段，打破沟通壁垒，让更多人以一种有趣、无压力的方式接触并学习手语。

### English
**Signs of Resistance** is an immersive HTML5 game that blends narrative storytelling with American Sign Language (ASL) interaction. In this project, players progress through the story not just by clicking a mouse, but by performing actual sign language gestures.

The goal of this project is to leverage technology to bridge communication gaps, allowing more people to experience and learn sign language in an engaging, stress-free environment.

---

## ✨ 核心功能 / Key Features

* **沉浸式叙事 (Immersive Storytelling)**: 结合情节发展，让手语练习不再枯燥。
* **实时交互 (Real-time Interaction)**: 基于浏览器的实时手语检测与反馈。
* **开源学习 (Open Source)**: 代码完全开源，欢迎开发者共同改进识别算法。
